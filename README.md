## ⚙️&nbsp; 기능
> ### 보드
1. 보드 추가
- 오른쪽 하단의 '+' 버튼을 클릭하면 보드 추가창이 뜹니다.
- 추가할 보드의 이름을 입력합니다.

2. 보드 수정/삭제
- 수정할 보드에 마우스 오버를 하면 버튼이 뜹니다.
<br/>

> ### 카드
1. 카드 추가
- 보드의 입력창에 메모를 입력하고 엔터를 쳐서 추가합니다.

2. 카드 수정/삭제
- 수정할 카드에 마우스 오버를 하면 버튼이 뜹니다.

3. 카드 재정렬 (드래그 앤 드랍)
- 카드를 다른 보드로 끌어 이동시킵니다.
